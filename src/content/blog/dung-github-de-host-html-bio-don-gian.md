```json
# Personal Bio Landing Page

## User Information

- Name: Alex Carter
- Profession: UI/UX Designer & Content Creator
- Short Bio: I design thoughtful digital experiences and create content that inspires creativity and learning.
- Status: 🟢 Available for freelance
- Portfolio: https://nhatnguyen.com
- LinkedIn: https://linkedin.com/in/example
- Behance: https://behance.net/example
- Dribbble: https://dribbble.com/example
- Instagram: https://instagram.com/example
- Email: hello@example.com

### Featured Projects
- Creative Portfolio — A curated collection of design work.
- Design System — A scalable UI kit for modern products.
- Creator Journal — A blog sharing design insights and creative processes.

### Skills
- UI/UX Design
- Product Design
- Branding
- Design Systems
- Figma
- Prototyping
- Content Strategy
- Creative Direction

---

## Goal

Create a premium, minimalist personal bio landing page using the information above. The design should feel inspired by Apple, Linear, and Vercel, with clean typography, generous whitespace, a monochrome color palette, rounded corners, and subtle premium animations.

## Sections

- Hero (Avatar, Name, Profession, Bio, Status)
- Social Links
- Featured Projects
- Skills
- Footer

## Technical Requirements

- Return ONLY one self-contained HTML file.
- Use only HTML, inline CSS, and vanilla JavaScript.
- No external libraries, frameworks, CDNs, or assets.
- Fully responsive (mobile-first).
- Clean, semantic, production-quality code.
- Smooth animations and polished interactions.
- Use SVG or CSS for icons/placeholders when needed.

## Result

The page should look modern, elegant, and realistic—as if it were a real personal website ready to deploy on GitHub Pages.
```
